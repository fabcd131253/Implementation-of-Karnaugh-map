此程式用於實現輸出 Karnaugh map 的圖示、prime implicants、essential prime implicants 及 variables 的 minimum minterms。
最多支持 4 個變數的輸入。若有超過一組 minimum minterm 時，只會輸出其中一項。輸入檔案名稱為 input.txt，輸出檔案名稱為 output.txt。

壓縮檔包含 kmap.java、mqc.java、petrick.java、FileIO.java，執行之主程式為 kmap.java。
程式於 Windows 10 上，使用 java 17 撰寫。

程式功能之敘述及詳細資料撰寫於壓縮檔中的 report.pdf 及 README.md 中，以繁體中文撰寫。
